# rock-paper-scissors

This is a program that allows user to play rock paper scissors with computer.

This is also my first JavaScript program built from scratch after reading and learning from TOP.
The aim of this program is to practice my problem solving skills, write out algorithm and psuedo-code, and convert those into a JavaScript program.

So I'm going to start out and write the algorithm below here:
    start the game
    ask user to select rock, paper or scissor
    get a random selection for computer
    compare user's selection and computer's selection
        return who's the winner
    display who's the winner
    repeat 5 times
